# Control de Stock y Ventas — Kiosco Escolar

Aplicación full-stack con **Next.js (TypeScript)**, **PostgreSQL** (levantable con **Docker**) y base de datos diseñada para:

- Registrar ventas durante cada **recreo** con flujo ultra rápido (modo venta).
- Mantener **trazabilidad** de stock (ingresos, salidas, ajustes/bajas).
- Generar al cierre un **ticket general** y reportes básicos.

## Objetivo del repositorio

Ofrecer un esqueleto con arquitectura clara (capas) y un plan paso a paso para implementar el MVP y evolucionar a producción.
